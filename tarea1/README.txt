/ -----------------------------------------------/

Debe utilizar command prompt (CMD) para correr el script.

python {nombre-archivo-python}.py < {nombre-archivo-prueba}.dat

Si no sirve el prefijo python utilizar:

python3 {nombre-archivo-python}.py < {nombre-archivo-prueba}.dat

/ -----------------------------------------------/

Ejemplo:
    python problema1.py < input-1.dat
    python problema2.py < input-1.dat

Si no sirve el prefijo python utilizar
    python3 problema1.py < input-1.dat
    python3 problema2.py < input-1.dat


Comandos preparados:

python problema1.py < input-1.dat
python problema1.py < input-2.dat
python problema1.py < input-3.dat

python problema2.py < input-1.dat
python problema2.py < input-2.dat
python problema2.py < input-3.dat
